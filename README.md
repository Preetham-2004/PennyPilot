🌍 Problem Statement
Managing personal finances can be overwhelming, especially with the flood of daily expenses and lack of visual insight. Many people struggle to track spending habits and make informed financial decisions.

PennyPilot is an AI-powered web app that simplifies expense tracking and enhances financial awareness through smart insights and elegant design.

💡 Solution – PennyPilot
PennyPilot is an intuitive expense tracking assistant that helps users:

🧠 Leverage AI to analyze and categorize expenses intelligently.

📊 Visualize spending trends with interactive, dynamic charts.

🔔 Receive personalized suggestions and alerts based on spending behavior.

📈 Understand where their money goes with a clear, beautiful dashboard.

🔄 Workflow / How It Works
1️⃣ User logs expenses through voice input or manual entry.
2️⃣ AI automatically categorizes the expense and suggests tags.
3️⃣ Real-time charts update to reflect spending in various categories.
4️⃣ The assistant provides daily/weekly insights and financial tips.
5️⃣ Optional goal tracking and streak features encourage consistency.
