# 🚀 PennyPilot - Personal Expense Tracker with Soothing AI
---

## 🌍 Problem Statement

Managing personal finances can be overwhelming and confusing, especially when tracking spending patterns, analyzing expenses, and staying within budgets.

---

## 💡 Our Solution - PennyPilot

PennyPilot is a calm, intelligent expense tracking assistant that helps users visualize their expenses, analyze spending habits, and stay financially healthy.

---

## 🌟 Key Features

- ✅ **Interactive, AI-powered dashboard** for financial insights  
- ✅ **Dynamic Charts** showing expense breakdowns  
- ✅ **Speech-to-Text** support for quick voice entry  
- ✅ **Beautiful, soothing UI** designed for mental wellness  
- ✅ **Expense categorization** & budget insights  
- ✅ **Offline support** and dark mode

---

## 🔄 Workflow

1. 🧾 User enters or speaks an expense  
2. 🤖 AI categorizes the expense  
3. 📊 Charts and insights update in real-time  
4. 🧘‍♂️ Optional mood-tracking and wellness prompts  

---

## 🛠️ Tech Stack

**Frontend:** React + Vite + Tailwind CSS  
**Backend:** Node.js / Express or FastAPI  
**Database:** MongoDB / SQLite  
**Extras:** Sentiment Analysis, Chart.js or Recharts  

---

## 🌱 Impact

- ✔️ Promotes financial awareness  
- ✔️ Reduces stress from money management  
- ✔️ Encourages healthier spending habits
